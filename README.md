# Jalur-Laut

**Jalur-Laut** adalah sebuah website pemesanan tiket kapal secara online yang dikembangkan sebagai proyek kerja tim untuk memenuhi salah satu tugas pada mata kuliah **Pemrograman Web Framework**.

Proyek ini bertujuan untuk menerapkan konsep pengembangan web modern berbasis framework, mulai dari perancangan arsitektur aplikasi, pemisahan frontend dan backend, hingga implementasi komunikasi antara client dan server.
